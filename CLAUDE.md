## Project

**DB Diff Tool - Project Context**

一个命令行数据库比对工具，用于数据库迁移/重构场景下的表结构和数据差异分析。

**核心价值：** 一条命令完成两个 MySQL 数据库的全面对比，生成清晰的 HTML 差异报告。

**Core Value:** **简单易用** — 一条命令完成对比，配置灵活，报告清晰。
