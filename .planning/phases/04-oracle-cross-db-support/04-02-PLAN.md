    1. Define MappingStatus type:
       ```go
       type MappingStatus string
       const (
           StatusSame       MappingStatus = "same"        // ✅ 相同
           StatusMapped     MappingStatus = "mapped"      // ⚠️ 已映射匹配
           StatusIncompatible MappingStatus = "incompatible" // ❌ 不兼容
       )
       ```

    2. Define MappingResult struct:
       ```go
       type MappingResult struct {
           Status      MappingStatus
           MappedType  string  // Mapped type (empty if StatusSame or incompatible)
           Description string  // Human-readable description
       }
       ```

    3. Define bidirectional type mapping tables (per 04-CONTEXT.md):
       ```go
       var mysqlToOracleTypeMap = map[string]string{
           "VARCHAR": "VARCHAR2", "CHAR": "CHAR", "TEXT": "CLOB", "BLOB": "BLOB",
           "INT": "NUMBER", "BIGINT": "NUMBER", "SMALLINT": "NUMBER", "TINYINT": "NUMBER(1)",
           "DATETIME": "DATE", "TIMESTAMP": "TIMESTAMP", "DATE": "DATE",
           "DECIMAL": "NUMBER", "FLOAT": "BINARY_FLOAT", "DOUBLE": "BINARY_DOUBLE",
       }

       var oracleToMySQLTypeMap = map[string]string{
           "VARCHAR2": "VARCHAR", "CHAR": "CHAR", "CLOB": "TEXT", "BLOB": "BLOB",
           "NUMBER": "DECIMAL", "BINARY_FLOAT": "FLOAT", "BINARY_DOUBLE": "DOUBLE",
           "DATE": "DATETIME", "TIMESTAMP": "TIMESTAMP",
       }
       ```

    4. Implement MapType(sourceType, targetType, sourceDriver, targetDriver string) MappingResult:
       - If sourceDriver == targetDriver and sourceType == targetType → StatusSame
       - If sourceDriver == "mysql" and targetDriver == "oracle": lookup mysqlToOracleTypeMap
       - If sourceDriver == "oracle" and targetDriver == "mysql": lookup oracleToMySQLTypeMap
       - Exact match in map → StatusMapped with MappedType
       - Not found → StatusIncompatible

    5. Implement AreTypesCompatible(sourceType, targetType, sourceDriver, targetDriver) bool:
       - Returns true if StatusSame or StatusMapped
       - Returns false if StatusIncompatible

       ```go
       func getDriver(driverName string) (database.Driver, error) {
           switch driverName {
           case "mysql":
               return &database.MySQLDriver{}
           case "oracle":
               return &database.OracleDriver{}
           default:
               return nil, fmt.Errorf("unknown driver: %s", driverName)
           }
       }
       ```

    3. Update Compare() function:
       - Replace `database.Connect(cfg.Source)` with:
         ```go
         sourceDriver, err := getDriver(cfg.Source.Driver)
         sourceConn, err := sourceDriver.Connect(cfg.Source)
         ```
       - Replace `database.Connect(cfg.Target)` similarly
       - Use `driver.FetchMetadata(conn, cfg.GetSchema())` with schema from GetSchema()

    4. Add type mapping to column comparison:
       - In compareColumn() or new compareColumnsWithMapping():
       ```go
       typeMapping := type_mapper.MapType(source.DataType, target.DataType, cfg.Source.Driver, cfg.Target.Driver)
       ```

    5. Extend ColumnModification struct:
